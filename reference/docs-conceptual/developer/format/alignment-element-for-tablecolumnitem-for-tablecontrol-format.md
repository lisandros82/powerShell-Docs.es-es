# <a name="alignment-element-for-tablecolumnitem-for-tablecontrol-format"></a>Elemento Alignment para TableColumnItem for TableControl (formato)

Define cómo se muestran los datos de una columna de la fila.

Elemento Configuration (Format) elemento ViewDefinitions (Format) elemento View (Format) elemento Tablecontrol ((Format) elemento TableRowEntries (Format) TableRowEntry (Format) elemento TableColumnItems (Format) elemento TableColumnItem (Format) Elemento Alignment para TableColumnItem (Format)

## <a name="syntax"></a>Sintaxis

```xml
<Alignment>AlignmentType</Alignment>
```

## <a name="attributes-and-elements"></a>Atributos y elementos

En las secciones siguientes se describen los atributos, los elementos secundarios y el elemento primario del elemento `Alignment`.

### <a name="attributes"></a>Atributos

Ninguna.

### <a name="child-elements"></a>Elementos secundarios

Ninguna.

### <a name="parent-elements"></a>Elementos primarios

|Elemento|Descripción|
|-------------|-----------------|
|[Elemento TableColumnItem (Format)](./tablecolumnitem-element-for-tablecolumnitems-for-tablecontrol-format.md)|Define una etiqueta, el ancho y la alineación de los datos de una columna de la tabla.|

## <a name="text-value"></a>Valor de texto

Especifique uno de los valores siguientes. (Estos valores no distinguen mayúsculas de minúsculas).

Left desplaza los datos mostrados en la columna hacia la izquierda. (Este es el valor predeterminado si no se especifica este elemento).

Desplaza a la derecha los datos mostrados en la columna.

Center centra los datos mostrados en la columna.
