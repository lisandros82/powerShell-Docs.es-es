# <a name="text-element-for-customitem-for-controls-for-view-format"></a>Elemento Text para CustomItem for Controls for View (formato)

Especifica el texto que se agrega a los datos mostrados por el control, como una etiqueta, corchetes para incluir los datos y espacios para aplicar sangría a los datos. Este elemento se utiliza al definir controles que se pueden usar en una vista.
