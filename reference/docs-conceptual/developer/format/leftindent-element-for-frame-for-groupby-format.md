# <a name="leftindent-element-for-frame-for-groupby-format"></a>Elemento LeftIndent para Frame for GroupBy (formato)

Especifica el número de caracteres que los datos se desplazan fuera del margen izquierdo. Este elemento se utiliza al definir cómo se muestra un nuevo grupo de objetos.
