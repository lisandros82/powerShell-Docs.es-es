# <a name="rightindent-element-for-frame-for-groupby-format"></a>Elemento RightIndent para Frame for GroupBy (formato)

Especifica el número de caracteres que los datos se desplazan fuera del margen derecho. Este elemento se utiliza al definir cómo se muestra un nuevo grupo de objetos.
