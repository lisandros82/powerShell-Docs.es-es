# <a name="cmdlet-development-guidelines"></a>Directrices de desarrollo del cmdlet

En los temas de esta sección se proporcionan instrucciones de desarrollo que puede usar para generar cmdlets correctos. Al aprovechar la funcionalidad común proporcionada por el tiempo de ejecución de Windows PowerShell y siguiendo estas instrucciones, puede desarrollar cmdlets sólidos con un esfuerzo mínimo y proporcionar al usuario una experiencia coherente. Además, reducirá la carga de pruebas porque la funcionalidad común no requiere una reprueba.
