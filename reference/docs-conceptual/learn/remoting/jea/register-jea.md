<span data-ttu-id="7e9a8-144">A continuación se muestra un ejemplo de configuración de DSC de un módulo de mantenimiento general del servidor.</span><span class="sxs-lookup"><span data-stu-id="7e9a8-144">Below is a sample DSC configuration for a general server maintenance module.</span></span> <span data-ttu-id="7e9a8-145">Supone que un módulo de PowerShell válido que contiene las funcionalidades de rol se encuentra en el recurso compartido de archivos `\\myfileshare\JEA`.</span><span class="sxs-lookup"><span data-stu-id="7e9a8-145">It assumes that a valid PowerShell module containing role capabilities is located on the `\\myfileshare\JEA` file share.</span></span>

```powershell
Configuration JEAMaintenance
{
    Import-DscResource -Module JustEnoughAdministration, PSDesiredStateConfiguration

    File MaintenanceModule
    {
        SourcePath = "\\myfileshare\JEA\ContosoMaintenance"
        DestinationPath = "C:\Program Files\WindowsPowerShell\Modules\ContosoMaintenance"
        Checksum = "SHA-256"
        Ensure = "Present"
        Type = "Directory"
        Recurse = $true
    }

    JeaEndpoint JEAMaintenanceEndpoint
    {
        EndpointName = "JEAMaintenance"
        RoleDefinitions = "@{ 'CONTOSO\JEAMaintenanceAuditors' = @{ RoleCapabilities = 'GeneralServerMaintenance-Audit' }; 'CONTOSO\JEAMaintenanceAdmins' = @{ RoleCapabilities = 'GeneralServerMaintenance-Audit', 'GeneralServerMaintenance-Admin' } }"
        TranscriptDirectory = 'C:\ProgramData\JEAConfiguration\Transcripts'
        DependsOn = '[File]MaintenanceModule'
    }
}
```
