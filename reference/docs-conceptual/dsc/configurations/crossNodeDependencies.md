# <a name="specifying-cross-node-dependencies"></a><span data-ttu-id="e34cd-103">Especificación de dependencias entre nodos</span><span class="sxs-lookup"><span data-stu-id="e34cd-103">Specifying cross-node dependencies</span></span>

<span data-ttu-id="e34cd-105">DSC proporciona recursos especiales, **WaitForAll**, **WaitForAny** y **WaitForSome**, que se pueden usar en configuraciones para especificar las dependencias en configuraciones de otros nodos.</span><span class="sxs-lookup"><span data-stu-id="e34cd-105">DSC provides special resources, **WaitForAll**, **WaitForAny**, and **WaitForSome** that can be used in configurations to specify dependencies on configurations on other nodes.</span></span> <span data-ttu-id="e34cd-106">El comportamiento de estos recursos es el siguiente:</span><span class="sxs-lookup"><span data-stu-id="e34cd-106">The behavior of these resources is as follows:</span></span>

## <a name="using-waitforxxxx-resources"></a><span data-ttu-id="e34cd-135">Uso de recursos WaitForXXXX</span><span class="sxs-lookup"><span data-stu-id="e34cd-135">Using WaitForXXXX resources</span></span>

<span data-ttu-id="e34cd-136">Cada recurso **WaitForXXXX** espera a que se completen los recursos especificados en el nodo especificado.</span><span class="sxs-lookup"><span data-stu-id="e34cd-136">Each **WaitForXXXX** resource waits for the specified resources to complete on the specified Node.</span></span>
<span data-ttu-id="e34cd-137">Otros recursos de la misma configuración pueden entonces *depender* del recurso **WaitForXXXX**  usando la clave **DependsOn**.</span><span class="sxs-lookup"><span data-stu-id="e34cd-137">Other resources in the same Configuration can then *depend on* the **WaitForXXXX** resource using the **DependsOn** key.</span></span>

<span data-ttu-id="e34cd-138">Por ejemplo, en la configuración siguiente, el nodo de destino está esperando que el recurso **xADDomain** finalice en el nodo **MyDC** con un número máximo de 30 reintentos, a intervalos de 15 segundos, antes de que el nodo de destino pueda unirse al dominio.</span><span class="sxs-lookup"><span data-stu-id="e34cd-138">For example, in the following configuration, the target node is waiting for the **xADDomain** resource to finish on the **MyDC** node with maximum number of 30 retries, at 15-second intervals, before the target node can join the domain.</span></span>

<span data-ttu-id="e34cd-139">De manera predeterminada, los recursos de **WaitForXXX** lo intentan una vez y después se produce un error.</span><span class="sxs-lookup"><span data-stu-id="e34cd-139">By default the **WaitForXXX** resources try one time and then fail.</span></span> <span data-ttu-id="e34cd-140">Aunque no es necesario, normalmente la intención será especificar un valor para **RetryCount** y otro para **RetryIntervalSec**.</span><span class="sxs-lookup"><span data-stu-id="e34cd-140">Although it is not required, you will typically want to specify a **RetryCount** and **RetryIntervalSec**.</span></span>

```powershell
Configuration JoinDomain
{
    Import-DscResource -Module xComputerManagement, xActiveDirectory

    Node myDC
    {
        WindowsFeature InstallAD
        {
            Ensure = 'Present'
            Name = 'AD-Domain-Services'
        }

        xADDomain NewDomain
        {
            DomainName = 'Contoso.com'
            DomainAdministratorCredential = (Get-Credential)
            SafemodeAdministratorPassword = (Get-Credential)
            DatabasePath = "C:\Windows\NTDS"
            LogPath = "C:\Windows\NTDS"
            SysvolPath = "C:\Windows\Sysvol"
        }
    }

    Node myDomainJoinedServer
    {
        WaitForAll DC
        {
            ResourceName      = '[xADDomain]NewDomain'
            NodeName          = 'MyDC'
            RetryIntervalSec  = 15
            RetryCount        = 30
        }

        xComputer JoinDomain
        {
            Name             = 'myPC'
            DomainName       = 'Contoso.com'
            Credential       = (Get-Credential)
            DependsOn        ='[WaitForAll]DC'
        }
    }
}
```
